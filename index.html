RailsGirls
Summer of Code

RailsGirls Summer of Code is a project that aims at lowering the barriers for students of newcomer workshops to participate in Open Source projects.

Following the footsteps of Google Summer of Code and Ruby Summer of Code students will be paid so they're free to work on Open Source projects for a few months. Open Source projects set goals and name mentors. Coaches help students to get started, keep motivated and be successful.

Unlike those programs the RailsGirls Summer of Code is not about pushing sophisticated, high-end code that very few people code write. It is about lowering barriers, connecting people, motivating newcomers and raising diversity.

We are currently aiming at projects running up to three months.

Do you run an Open Source project?

Please consider registering here. Any project is elegible as long as it's open source.

Possible goals can be anything from adding features and writing actual code to improving the documentation to doing html/css or design work. It doesn't need to be "just code", instead it actually might help if students could bring in their background (if they're designers, do some design work, if they're good at writing, write something, etc.)

Also, consider setting a number of small goals or follow-up goals so you can adjust to students if need be.

Please name a (main) mentor who's up to support the student throughout the project. Projects might share work on this and have "secondary mentors" but there should be one, main personal mentor.

Are you a coach?

Please consider registering here.

You should bring enough time to support the student throughout the project. Ideally you'd be able to provide a workplace throughout the project, so the student you're coaching can work on-site, next to you.
