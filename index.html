Welcome to the second edition!

Last year Rails Girls Summer of Code started for the first time and has been a blast. This year, we will be back and make another Summer full of code happen with you!

Rails Girls Summer of Code offers three-months full time scholarships to Rails Girls students. This makes it possible for the students to get deeper into coding and Open Source. This initiative is about about learning by doing - expand the knowledge and coding skills by contributing to a great Open Source project.

The Rails Girls Summer of Code program starts 1st of July 2014.

Timeline

  • 14.03.: Applications open
  • March: Crowdfunding start
  • 02.05.: Close applications!
  • 04.05.: Rank applications
  • 06.05.: Acceptance letters
  • June: Preparations
  • 1.07.: Start RGSoC 2014!
